# Event Images Folder

Place your event images in the corresponding subfolders:

## Current Event Folders:
- **cruise/** - Cruise event photos
- **qwic-welcome/** - QWIC Welcome Home Night photos
- **ai-collective/** - AI Collective event photos
- **orientation-2025/** - Orientation 2025 photos
- **resume-roast/** - Resume Roast event photos
- **group-shots/** - COMPSA Group Shots photos
- **formal-2025/** - 2025 Formal photos
- **merch-shoot/** - 2025 Merch Shoot photos
- **basketball/** - Basketball Tournament photos
- **fall-night/** - 2025 Fall Night Out photos
- **soccer-2024/** - 2024 Soccer Tournament photos

## How to Add Images:
1. Copy your images into the appropriate folder
2. Update `src/components/Gallery.js` with the filenames
3. Refresh the browser

See HOW_TO_ADD_IMAGES.md in the project root for detailed instructions.
